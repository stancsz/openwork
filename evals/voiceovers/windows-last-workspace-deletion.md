# windows-last-workspace-deletion - Last workspace stays deleted on Windows

1. With multiple workspaces in the sidebar, I remove one and OpenWork keeps the remaining workspace ready to use.

2. I remove the final workspace, and OpenWork shows a clean empty workspace screen with nothing left behind.

3. After waiting, I collapse the empty sidebar. The screen stays quiet with no stale workspace activity and no unexpected model picker.

4. I restart OpenWork and open Create Workspace. The empty workspace list survived the restart, and OpenWork is ready for whatever comes next.
