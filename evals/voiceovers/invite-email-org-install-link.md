# invite-email-org-install-link — The invitation email uses the organization's installer

1. Alex invites Maya from Acme Robotics, and the Members page immediately shows her invitation as pending.

2. Maya opens the real invitation email and sees both ways forward: accept the invite, or download the desktop app through Acme Robotics' own install link instead of a generic marketing page.

3. On her Windows computer, that email link opens the Acme Robotics installer page and its Windows action resolves to a real executable without carrying the private organization token into any fallback URL.
