# desktop-rename-groups — Manage session groups from the sidebar

1. The session sidebar shows each group’s session count aligned flush right, with no empty spacing reserved for hidden actions.

2. Hovering over a group row replaces its count with an ellipsis. Hovering over the ellipsis expands it to reveal create session, rename group, and delete group actions.

3. Clicking the plus icon immediately creates a new session inside that group.

4. Clicking the pencil opens **Rename Group**, with the current name pre-filled and **Cancel** and **Save** actions. Saving updates the group name in the sidebar.

5. Clicking the trash icon opens **Delete Group**. I choose where its sessions should move—another existing group or **Ungrouped**—then cancel or confirm.

6. After confirmation, the group disappears and its sessions appear in the selected destination.
