# sen-api-desktop-versions — Organization owners can choose every published desktop version their server supports

1. In Org Cloud Settings under General and Allowed Desktop Versions, the minimum version is now v0.17.0.

2. Opening the version list shows published GitHub release tags beginning with v, sorted newest first, including every version between the configured minimum and maximum.

3. Versions newer than the maximum remain visible but disabled, with the note “Upgrade server to allow this version.”

4. The version list is capped at 400 pixels and scrolls, keeping the settings page compact.
